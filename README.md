# Hotel-Management-using-spring

The Hotel Management System is a web-based application designed to facilitate the ordering and delivery of food from various restaurants. Implemented features such as user validation, menu browsing, and order processing. Integrated with a backend database for efficient data management. Employed endpoints for seamless communication between the frontend and backend. Achieved a responsive and intuitive UI for enhanced user experience. Admin can approve and block Hotels. Admin is responsible to maintain everything.   Hotels can upload products, update product, delete products. User can book a order, can view his previous food orders.
